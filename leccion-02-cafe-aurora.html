El problema

La diferencia entre un resultado genérico y uno profesional está en cómo escribes el prompt.

✕  Prompt vago
"crea una página web"
Resultado: página genérica, sin personalidad ni estructura. La IA adivina lo que quieres.
VS
✓  Prompt completo
"Crea el diseño de una landing page profesional para una cafetería moderna llamada Café Aurora. La página debe tener un estilo elegante, cálido y minimalista..."
Resultado: experiencia visual completa, con secciones definidas, estilo coherente y orientada a convertir.

La fórmula del buen prompt

01Qué quiero crearEl tipo de producto, su nombre y categoría
02Para quién esLa audiencia o el cliente al que va dirigido
03Cómo debe verseEstilo visual, colores, tipografía y tono
04Qué debe incluirLas secciones, funciones o elementos clave
05Qué objetivo cumplirEl resultado o la acción que se busca lograr

01 Qué quiero crear
Crea el diseño de una landing page profesional para una cafetería moderna llamada Café Aurora.
03 Cómo debe verse
La página debe tener un estilo elegante, cálido y minimalista, con una estética premium pero cercana. Usa colores suaves, tipografía moderna y una composición visual limpia.
