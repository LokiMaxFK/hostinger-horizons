Los 5 pasos

Así se construye un prompt claro y útil.

01

Define el proyecto

Landing page, app web, dashboard o sitio web.

02

Da contexto

Indica para quién es y qué problema resuelve.

03

Marca el estilo

Moderno, minimalista, premium, tecnológico o cálido.

04

Estructura la página

Hero, beneficios, testimonios, CTA, FAQ y footer.

05

Explica el objetivo

Vender, captar leads, generar confianza o reservar.

Primero piensa el proyecto; después escríbelo.
